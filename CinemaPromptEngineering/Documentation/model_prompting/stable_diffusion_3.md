# Stable Diffusion 3 Prompting Guide

Category: Image

## Official Guidance Highlights
- Use clear natural language and define style first.
- Specify subject and action, then composition, lighting, and color.
- Use camera and framing terms (close-up, wide-angle, bird's-eye view).
- For text, use short phrases in quotes.
- Negative prompts can refine unwanted elements.

## Recommended Prompt Structure
1. Style or medium
2. Subject + action
3. Composition and framing
4. Lighting and color
5. Technical cues (lens, viewpoint)
6. Text in quotes (optional)
7. Negative prompt (optional)

## Positive Examples
- "Photography, wide shot of a modern cabin in snow, blue hour light, warm interior glow, crisp detail."
- "Illustration, a red fox in tall grass, soft morning light, muted palette, shallow depth of field."

## Negative Examples (What to Avoid)
- "Make it cool" (no visual details)
- "No blur, no noise, no artifacts" (use a negative prompt field instead)

## Notes
- Use composition terms to control framing and perspective.
