# Leonardo AI Prompting Guide

Category: Image

## Official Guidance Highlights
- Start with the main subject and key attributes.
- Add environment, lighting, and style for control.
- Clear, specific wording yields better adherence.

## Recommended Prompt Structure
1. Subject + key attributes
2. Action or pose
3. Environment and lighting
4. Style or medium
5. Optional text in quotes

## Positive Examples
- "Vintage travel poster of a coastal town, pastel palette, clean vector shapes, subtle paper texture."
- "Studio portrait of a chef, soft window light, warm tones, shallow depth of field."

## Negative Examples (What to Avoid)
- "Make it epic and cool" (vague)
- "Photo, watercolor, oil, 3D" (conflicting styles)

## Notes
- Keep the core subject first for better adherence.
