# Qwen-VL Prompting Guide

Category: Video

## Best Practices
- Qwen-VL is a vision-language model; state the task and expected output clearly.
- For video inputs, specify time ranges or key frames in the request.
- Ask for structured outputs if needed.

## Common Pitfalls
- Assuming it is a text-to-video generator.
- Vague requests like analyze this without a goal.

## Short Example
```text
Analyze the provided video: summarize the main action in 2 sentences and list any text you see on screen.
```
