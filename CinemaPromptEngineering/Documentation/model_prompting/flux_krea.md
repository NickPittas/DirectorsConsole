# Flux Krea Prompting Guide

Category: Image

## Official Guidance Highlights
- Use structured prompts: [art style] [subject] [scene] [lighting] [color].
- Combine text prompts with style references for stronger control.
- Use multiple references and adjust weights to balance prompt vs style.
- Iterate with vary/regenerate to refine.

## Recommended Prompt Structure
1. Art style
2. Subject
3. Scene/environment
4. Lighting
5. Color palette

## Positive Examples
- "A linocut illustration of a forest clearing, soft natural light, warm earthy tones"
- "Editorial photo of a ceramic vase on a wooden table, diffused window light, muted beige palette"

## Negative Examples (What to Avoid)
- "Nice style, make it better" (no concrete visual descriptors)
- "Photo, watercolor, oil, 3D" (conflicting styles)

## Notes
- If using references, lower strength preserves prompt details; higher strength favors the reference.
